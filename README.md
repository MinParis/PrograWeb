# Ejercicios de Desarrollo Web

## Descripción

Actividades de Programacion web , Repositorio con las 15 actividades realizadas desde visual con html, css y javascript

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap (en algunas)

## Ejercicios

### Ejercicio 1 – Mi Primera Página HTML
Creación de una página web básica utilizando la estructura mínima de HTML5 y mostrando el mensaje "¡Hola Mundo!".

### Ejercicio 2 – Párrafos y Formato Básico
Uso de etiquetas para dar formato al texto, incluyendo párrafos, texto en negrita, cursiva y saltos de línea.

### Ejercicio 3 – Enlaces Simples
Creación de enlaces externos utilizando la etiqueta `<a>` y los atributos `target="_blank"` y `target="_self"`.

### Ejercicio 4 – Imágenes Básicas
Inserción de imágenes mediante la etiqueta `<img>`, utilizando atributos como `src`, `alt` y `width`.

### Ejercicio 5 – Combinando Texto y Enlaces
Integración de imágenes, texto y enlaces dentro de una misma página para reforzar el uso de los elementos básicos de HTML.

### Ejercicio 6 – Secciones Semánticas
Uso de las etiquetas semánticas `<header>`, `<section>` y `<footer>` para organizar correctamente la estructura del documento.

### Ejercicio 7 – Listas Anidadas
Desarrollo de una receta utilizando listas ordenadas y desordenadas, aplicando listas anidadas para organizar la información.

### Ejercicio 8 – Horario de Clases
Creación de una tabla con el horario semanal utilizando `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>` y `<td>`, incorporando estilos con CSS.

### Ejercicio 9 – Formulario de Contacto
Diseño de un formulario de contacto utilizando elementos de HTML5 como `<form>`, `<input>`, `<textarea>`, `<select>`, `<fieldset>` y `<button>`, aplicando validaciones y estilos mediante CSS.

### Actividad 10 – Conversor de Temperatura
Aplicación web que convierte grados Celsius a Fahrenheit utilizando JavaScript. Incluye validaciones para comprobar que el campo no esté vacío y que el valor ingresado sea numérico, mostrando el resultado en un campo de solo lectura.

### Actividad 11 – Conversor de Distancia
Desarrollo de una aplicación que convierte kilómetros a millas mediante JavaScript. El sistema valida la información ingresada y muestra el resultado en un campo de solo lectura.

### Actividad 12 – Conversor de Moneda
Aplicación web que convierte una cantidad de pesos mexicanos (MXN) a dólares estadounidenses (USD), validando que el dato ingresado sea un número positivo antes de realizar la conversión.

### Actividad 13 – Verificador de Edad para Votar
Aplicación que determina si una persona puede votar de acuerdo con la edad ingresada. El sistema valida la información y muestra el resultado en un campo de solo lectura.

### Actividad 14 – Operaciones con Arreglos
Aplicación que recibe una lista de números separados por comas y calcula el número mayor, el número menor y el promedio utilizando arreglos y métodos propios de JavaScript.

### Actividad 15 – Gestión de Estudiantes con Objetos
Desarrollo de una aplicación para registrar estudiantes mediante objetos de JavaScript. Calcula el promedio de calificaciones e identifica al estudiante con la mayor y menor calificación.
